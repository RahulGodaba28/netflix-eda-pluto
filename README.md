# Netflix EDA — Pluto Academy AI/ML Internship

Exploratory Data Analysis on the Netflix Movies & TV Shows dataset.

## Dataset
[Netflix Movies and TV Shows — Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)

## Tools Used
Python · Pandas · NumPy · Matplotlib · Seaborn · Google Colab

## What's Inside
- Data loading, inspection & cleaning
- 5 EDA questions answered with Pandas
- 7 visualizations (bar, pie, line, histogram, heatmap)
- 5 business insights report

## How to Run
1. Open the `.ipynb` file in Google Colab
2. Upload `netflix_titles.csv` from Kaggle
3. Run all cells top to bottom
